# AltScore AI

## Vibecoded  using AI-assisted development

AI-powered alternative credit scoring for users without formal financial history.

## Problem
Millions of users (gig workers, informal sector) lack access to credit due to no formal credit history.
## Solution

AltScore AI uses alternative behavioral signals:

* UPI transaction patterns
* Mobile recharge frequency
* Location stability
* App usage behavior
* P2P transfers

## How it works

Input: User behavioral data
Output:

* Credit Score (300–900)
* Risk Level (Low / Medium / High)
* Behavioral Insights

## Example Output

User Profile: Gig worker with inconsistent income

Score: 682
Risk: Medium

Insights:

* Stable transaction frequency
* Moderate income volatility
* Consistent location patterns
